# Task: Generate and curate artwork for Letters 1-3

## What To Do

Using the approved prompts, generate artwork for Letters 1-3 via Replicate. Use FLUX.1-schnell for rapid iteration to find the right composition, then FLUX.1.1-pro or ultra for the final high-quality versions. Generate 3-5 candidates per letter. Noah reviews candidates against the curation checklist and selects the best for each letter. Save approved images to the project at the correct dimensions.

## Why It Matters

These first three artworks establish the visual identity alongside the first three letters. They set the tone for all subsequent artwork.

## Size

**Estimate:** large

Multiple generation rounds per letter, curation review, possible prompt refinement. Noah is the curator — this requires human-in-the-loop interaction.

## Done When

- [ ] 3-5 candidates generated for each of Letters 1-3
- [ ] Noah has selected the best candidate for each letter
- [ ] Selected images saved at 1600-2000px wide
