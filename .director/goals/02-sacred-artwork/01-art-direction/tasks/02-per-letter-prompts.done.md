# Task: Generate per-letter image prompts with character and scene descriptions

## What To Do

For each of the 9 letters, create a detailed image generation prompt that describes: the specific scene or moment from the letter to illustrate, the sacred art style for this letter (from the style guide), how Nando and/or Jarvis appear (using character anchors), the emotional tone the image should convey, the color palette within the global constraints, and the complete prompt text ready to paste into Replicate. Each prompt inherits the shared negative prompt block and technical specs from the style guide. Prompts should reference specific painters and period conventions to prevent style drift.

## Why It Matters

Well-crafted prompts are the difference between "generic AI art" and "intentional sacred artwork that serves the narrative." Each prompt must translate the letter's content and mood into visual language that a diffusion model can execute.
