# Goal 3: Readers experience the full story as an immersive, cinematic scroll

## What Success Looks Like

The website at isweartodog.com delivers the complete story as a single-page dark atmospheric scroll. Letters fade in as the reader scrolls, sacred artwork accompanies each letter, floating dot navigation tracks position, and the whole experience feels like reading an ancient secret by candlelight. It works beautifully on desktop and mobile, respects reduced-motion preferences, meets WCAG 2.1 AA, and scores 90+ on Lighthouse mobile. The code is clean, simple, and worth showing off in a public repo.

## Steps

1. **Step 1: Project setup and theme** -- Next.js 16 project with dark theme, fonts, and MDX configured
2. **Step 2: Component architecture** -- All UI components built with correct client/server boundaries
3. **Step 3: Content integration** -- Letters, artwork, and frame narrative assembled into the page
4. **Step 4: Accessibility and motion safety** -- WCAG compliance, reduced-motion support, semantic HTML
5. **Step 5: Performance tuning** -- Lighthouse 90+ mobile, optimized images, ambient effects
